# 2026-07-07 管理画面 UI/UX 改善

## 概要

既存管理画面（/login, /admin, /admin/persons, /admin/channels）のデザイン・UX・一貫性・
アクセシビリティ（WCAG 2.1 AA 目標）のレビューと改善実装。依存パッケージの追加なし。

## 最重要の発見・修正

**Tailwind v4 が `tailwind.config.ts` を読んでいなかった**。v4 の `@import "tailwindcss"` は
v3 形式の JS config を自動で読まないため、`bg-primary` 等のカスタム色ユーティリティが未生成、
`darkMode: 'class'` も未適用だった（プライマリボタンが無色、`dark:` バリアント全滅）。
→ `globals.css` に `@config "../../tailwind.config.ts"` を追加して解消。
併せて config が参照していた未定義の `--chart-1..5` / `--sidebar-*` CSS 変数を light/dark 両方に補完。
