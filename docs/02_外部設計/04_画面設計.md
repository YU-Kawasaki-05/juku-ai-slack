# 画面設計 — juku-ai-slack-bot (管理画面)

## 1. デザイン方針

- UIフレームワーク: shadcn/ui + Tailwind CSS [仮決定]
- レスポンシブ: デスクトップファースト（管理画面のため。モバイル対応は任意）
- アクセシビリティ: WCAG 2.1 AA準拠を目標
- 言語: 日本語UI
- カラー: Tailwindデフォルトカラー（ニュートラル系）

---

## 3. 画面詳細

### SCR-01: ログイン
**URL**: `/login`
**関連機能**: FR-13
**関連EP**: -（Supabase Auth SDK直接呼び出し）

```
┌─────────────────────────────────┐
│                                 │
│    [🎓 juku-ai-bot 管理画面]    │
│                                 │
│  メールアドレス                  │
│  [________________________]     │
│                                 │
│  パスワード                      │
│  [________________________]     │
│                                 │
│  [  ログイン  ]                  │
│                                 │
│  （エラー時）                    │
│  ⚠ メールアドレスまたはパスワード │
│    が正しくありません            │
└─────────────────────────────────┘
```

**要素一覧**:
| 要素 | 種別 | 必須 | バリデーション |
|------|------|------|--------------|
| メールアドレス | input[type=email] | ○ | メール形式 |
| パスワード | input[type=password] | ○ | 1文字以上 |
| ログインボタン | button[type=submit] | - | - |

**インタラクション**:
- ログイン成功 → `/dashboard` へリダイレクト
- ログイン失敗 → エラーメッセージ表示（入力は保持、パスワードはクリア）

**ローディング**: ログインボタンdisabled + スピナー

### SCR-06: チャンネル紐付け作成・編集
**URL**: `/bindings/new`, `/bindings/:id`
**関連機能**: FR-15
**関連EP**: EP-08, EP-09

**要素一覧**:
| 要素 | 種別 | 必須 | 備考 |
|------|------|------|------|
| Slack チャンネルID | input[text] | ○ | 新規作成時のみ入力可 |

| 生徒 | select | ○ | persons一覧から選択 |
| デフォルトレポート | select | NO | reports一覧から選択 |
| ステータス | toggle | ○ | active / inactive |

### SCR-12: エラー詳細
**URL**: `/errors/:id`
**関連機能**: FR-17
**関連EP**: EP-17

**要素**:
- エラーコード・severity・発生日時
- 生徒名・チャンネル名・スレッドURL（クリックでSlack開く）
- ユーザーへ返した文言
- 内部エラー詳細（raw_errorはMVP非表示）
- [対応済みにする] ボタン
- メモ欄（テキストエリア + 保存ボタン）

---

## 4. 共通コンポーネント一覧

| コンポーネント | 用途 |
|-------------|------|
| DataTable | 一覧テーブル（ページネーション・フィルタ付き） |
| StatusBadge | active/inactive/draft/published の視覚表現 |
| MarkdownEditor | レポート編集用エディタ（textarea or rich editor [仮決定]） |
| MarkdownRenderer | レポート表示用レンダラー |
| ConfirmDialog | 破壊的操作前の確認ダイアログ |
| LoadingSpinner | ローディング状態 |
| ErrorAlert | エラーメッセージ表示 |
| WarningBanner | Embedding再生成警告等 |

---

## 5. 状態管理

- サーバーステート: React Query (TanStack Query) または SWR [仮決定]
- フォームステート: React Hook Form + Zod
- 認証ステート: Supabase Auth クライアント（useSession）
