# FR-19: 会話ログ閲覧

## 概要

Slackスレッド単位の会話を管理画面から閲覧できる。MVP後に実装。

## 主要機能

- スレッド一覧表示
- 生徒別・チャンネル別・期間フィルタ
- 画像有無・使用モデル・token数・エラー有無での絞り込み
- 各スレッドのメッセージ詳細表示
